Personal Projects

I will be posting updates on my personal projects as I work on them. Currently in the design process are:

Desktop Robotic Gardener

The goal of this project is to create a tabletop hydroponic system which can grow plants without any human interaction. The sensors will monitor and be able to adjust water flow, light, temperature and pH levels. I have been awarded a grant through Northeastern University's IDEA Fund to assist with the development costs.

Glove controls for iRobot's Create 2

This arduino based controller will connect to a Roomba Create 2 over bluetooth serial controls. The drive controls will be located in a glove the user wears in order to adjust speed and turn. The goal of this project is to build an R2D2 unit on top of the Create 2 and be able to drive it with "the force".

Polygraph

Earlier this year I designed a polygraph as an introduction to biosensors and for a class on deception at Splash for Northeastern University's Program for Teaching by Undergraduates (NEPTUN) with Criminal Justice and Psychology student Alexa Lambros. The polygraph originally consisted of a pulse sensor, a galvanic skin response sensor, and a breathing rate monitor, but the breathing rate monitor was removed because we didn't think it would be a good idea to tie up high school students. The second part of this project was learning to accurately read a polygraph, which I was able to do approximately fifty percent of the time. Since the completion of this project, it has been disassembled so the parts can be reused.
